##Descripción del Proyecto

Este proyecto realiza un análisis exploratorio de datos completo sobre las ventas de cuatro tiendas virtuales de Alura Store Latam. El objetivo es extraer insights valiosos sobre el rendimiento del negocio a través de la manipulación y visualización de datos con Python.

##Objetivos del Análisis

Calcular la facturación total de cada tienda
Identificar las categorías de productos más vendidas
Analizar la satisfacción del cliente mediante calificaciones
Encontrar los productos más y menos vendidos
Evaluar los costos de envío por tienda

##Tecnologías Utilizadas

- **Python 3.8+**
- **Pandas** - Manipulación y análisis de datos
- **Matplotlib** - Visualización de datos
- **Seaborn** - Visualización estadística
- **Google Colab** - Entorno de desarrollo
